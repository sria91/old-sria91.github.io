Project Highlights

Implementation and Validation of Vision-based Ground Target Geo-Localization for Miniature Aerial System, 9.5 months
Environment: MATLAB, Visual Studio, C++, CMake, Git
The thesis presented a computer vision based algorithm for accurate geo-localization of a ground target from an aerial image captured by the Miniature Aerial System (MAS).

  • The user identifies and selects the ground target in the image taken by the camera on board Miniature Aerial Vehicle (MAV).
  • Geographic co-ordinates of the ground target are then calculated using the configuration data including: MAV position, attitude and the camera parameters.
  • A method for generating synthetic aerial image for any given pose configuration of the MAV and the camera is implemented.
  • The method has been implemented on MATLAB and tested with a custom build of FlightGear Flight Simulator.

Design and Development of Robot Mechanisms, Vision and Control Algorithms, 13 months
Environment: MATLAB, C, C++, SDL, GStreamer, Embedded Linux, GNU Make and Makefile, TI DVSDK
The project was sponsored by DST, Govt. of India. [video]

  • Setting up Angstrom Linux on BeagleBoard-xM ARM embedded platform for image and video processing applications

  • Development, Implementation and Optimization of target tracking algorithms on BeagleBoard-xM
  • Global Positioning System (GPS) and Inertial Measurement Unit (IMU) fusion for localization

Autonomous Mobile Robot Control, Localization and Path Planning, 16 months
Environment: MATLAB, Linux, Eclipse, C++, AVR, EAGLE PCB Design
Developed a 2D simulation environment in MATLAB for simulating a differential drive mobile robot. [videos]

  • The robot is capable of autonomously navigating to any desired goal location in a known static indoor environment by selecting an optimal and feasible path.
  • Ultrasonic range finders perceive the environment and the robot is able to localize using Extended Kalman Filter (EKF).
  • The path to be traversed by the robot is obtained from an artificial intelligence based path planning algorithm.
  • The robot motion control is achieved by using a kinematics based controller.
